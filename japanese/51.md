NIP-51
======

リスト
-----

`ドラフト` `オプション`

この NIP は、ユーザーが作成できるものごとのリストを定義します。リストには何でも参照を含めることができ、これらの参照は**公開**または**非公開**にすることができます。

リスト内の公開アイテムはイベントの `tags` 配列で指定され、非公開アイテムは JSON 配列で指定されます。この JSON 配列はイベントの `tags` 配列の構造を模倣していますが、文字列化され、[NIP-04](04.md) と同じスキームを使用して暗号化されます（共有キーは作成者の公開鍵と秘密鍵を使用して計算されます）。そして、`.content` に保存されます。

既存のリストに新しいアイテムを追加する場合、クライアントはそれらを時系列順に保存するために、リストの末尾に追加する必要があります（SHOULD）。

## リストの種類

## 標準リスト

標準リストはパラメータ化されていない置換可能イベントを使用します。つまり、ユーザーは各種類のリストを1つしか持つことができません。これらは特別な意味を持ち、クライアントはユーザーのプロフィールやブラウジング体験を強化するためにそれらに依存する場合があります。

例えば、_ミュートリスト_ には、ユーザーがフィードで見たくないスパマーや悪質な行為者の公開鍵を含めることができます。また、迷惑な通知を受け取りたくない相手の公開鍵も含めることができます。

| 名前              | 種類  | 説明                                                         | 予想されるタグアイテム                                                            |
| ----------------- | ----- | ------------------------------------------------------------ | --------------------------------------------------------------------------------- |
| ミュートリスト    | 10000 | ユーザーがフィードで見たくないもの                           | `"p"` (公開鍵), `"t"` (ハッシュタグ), `"word"` (小文字の文字列), `"e"` (スレッド) |
| ピン留めノート    | 10001 | ユーザーがプロフィールページで紹介したいイベント             | `"e"` (種類:1 のノート)                                                           |

| コミュニティ      | 10004 | ユーザーが所属する [NIP-72](72.md) コミュニティ              | `"a"` (種類:34550 のコミュニティ定義)                                             |
| 公開チャット      | 10005 | ユーザーが参加している [NIP-28](28.md) チャットチャンネル    | `"e"` (種類:40 のチャンネル定義)                                                  |

## セット

セットは、明確に定義された意味を持つリストで、それらに依存するクライアントの機能性とUIを強化することができます。標準リストとは異なり、ユーザーは各種類のセットを複数持つことが想定されているため、それぞれに異なる `"d"` 識別子を割り当てる必要があります。

例えば、_リレーセット_ はドロップダウンUIで表示され、ユーザーがイベントを公開するリレーや、イベントへの返信を読み取るリレーを切り替えるオプションを提供できます。_キュレーションセット_ は、アプリが他のユーザーによって作成された、異なるトピックにタグ付けされたキュレーションを紹介するために使用できます。

メインの識別子である `"d"` タグの他に、セットはオプションで `"title"`、`"image"`、`"description"` タグを持つことができ、これらはUIを強化するために使用できます。

| 名前                  | 種類  | 説明                                                                                     | 予想されるタグアイテム                                                                                                         |
| --------------------- | ----- | ---------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| フォローセット        | 30000 | クライアントが様々な状況で確認できるユーザーのカテゴリ分けされたグループ                 | `"p"` (公開鍵)                                                                                                                 |
| リレーセット          | 30002 | ユーザーが様々な操作中に簡単に選択できるユーザー定義のリレーグループ                     | `"relay"` (リレーURL)                                                                                                          |
| ブックマークセット    | 30003 | ユーザー定義のブックマークカテゴリ（ブックマークを分類して別々のグループにする必要がある場合） | `"e"` (種類:1 のノート), `"a"` (種類:30023 の記事), `"t"` (ハッシュタグ), `"r"` (URL)                                          |
| キュレーションセット  | 30004 | ユーザーが興味深いまたは同じカテゴリに属すると選んだ記事グループ                         | `"a"` (種類:30023 の記事), `"e"` (種類:1 のノート)                                                                            |
| キュレーションセット  | 30005 | ユーザーが興味深いまたは同じカテゴリに属すると選んだ動画グループ                         | `"a"` (種類:34235 の動画)                                                                                                      |
| 興味セット            | 30015 | 「ハッシュタグ」の集まりで表される興味のトピック                                         | `"t"` (ハッシュタグ)                                                                                                           |
| 絵文字セット          | 30030 | カテゴリ分けされた絵文字グループ                                                         | `"emoji"` ([NIP-30](30.md) 参照)                                                                                               |
| リリース成果物セット  | 30063 | ソフトウェアリリースのファイルグループ                                                   | `"e"` (種類:1063 の[ファイルメタデータ](94.md)イベント), `"i"` (アプリケーション識別子、通常は逆ドメイン表記), `"version"`     |

## 非推奨の標準リスト

一部のクライアントは過去にこれらのリストを使用していましたが、上記の[標準フォーマット](#標準リスト)への移行を進めるべきです。

| 種類  | "d" タグ        | 代わりに使用するもの                   |
| ----- | --------------- | -------------------------------------- |
| 30000 | `"mute"`        | 種類 10000 の _ミュートリスト_         |
| 30001 | `"pin"`         | 種類 10001 の _ピン留めリスト_         |
| 30001 | `"bookmark"`    | 種類 10003 の _ブックマークリスト_     |
| 30001 | `"communities"` | 種類 10004 の _コミュニティリスト_     |
